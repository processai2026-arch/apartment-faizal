## New Features Added (Beyond ₹15,000 Scope)

| # | Feature | Description |
|---|---------|-------------|
| 1 | Super Admin — Tenant Base Structure | 4th login tier for platform management. Create organisations, assign secretary credentials, set billing plan and features in one flow. |
| 2 | Multi-Organisation Support & Secretary Sub-Role | One platform manages multiple apartments. Secretary sub-role with configurable per-module permissions. |
| 3 | Suspend List | Admin can suspend/unsuspend user accounts. Suspended users cannot log in. |
| 4 | Documents, Medical & Name Transfer | Document storage with expiry tracking and PDF download. Medical report upload/download. Name transfer request management. |
| 5 | WhatsApp Integration (Free) | Built-in wa.me share links across all modules — no paid API. Templates for visitor, complaint, maintenance, announcement, invoice and more. |
| 6 | Vendor Reviews & Vendor Booking | Tenants browse, book, and rate vendors. Admin sets vendor ratings. Tenants can recommend new vendors. |
| 7 | Announcements, Community Events & Emergency Contacts | Admin publishes content; tenants receive it with glowing banner, event registration, and emergency contact visibility. |
| 8 | Business Ads & Ad Billing | Ad listings with click/impression tracking, ad packages, billing, renewal reminders and revenue analytics. |
| 9 | Daily Operations, Expense Tracking, Inventory & Assets | Daily task logs, office expense tracking, inventory movement, and individual asset checkout/audit system. |
| 10 | Reports — GST, Audit Log & Suspend List | GST and compliance reports, full audit log, suspended user list. CSV and PDF export for all reports. |
| 11 | AMC & DG Maintenance | Annual Maintenance Contract tracking and Diesel Generator service/fault log management. |
| 12 | Home Automation Integration | Home Assistant integration for smart device control. Tenant "My Home" page for personal device management. |
| 13 | CCTV Integration per Block | Camera registration per block/zone, health monitoring, event and snapshot logging. RTSP-ready architecture for future live stream. |
| 14 | Lift Fault Alert & Email Alerts | Lift and critical system fault events logged and notified. Email alert infrastructure for real-time admin notification. |
