## New Features Added (Beyond ₹15,000 Scope)

| # | Feature | Description |
|---|---------|-------------|
| 1 | **Super Admin — Tenant Base Structure** | 4th login tier for SaaS-level platform management. Create and manage multiple apartment organisations. Each org gets its own secretary, feature set, and billing plan — all from one super admin portal. |
| 2 | **Multi-Organisation Support & Secretary Sub-Role** | One platform instance manages multiple apartments. Secretary is an admin sub-role with configurable module-level permissions (can_view / can_edit per module). Secretary account auto-created during organisation provisioning. |
| 3 | **Suspend List** | Admin can suspend user accounts without deleting them. Suspended users cannot log in. Suspend/unsuspend from Security Management page. |
| 4 | **Documents, Medical & Name Transfer** | Document Management — upload, store, track expiry, and download PDFs per office/unit. Medical Reports — upload and download staff/resident medical records. Name Transfer — ownership/tenancy transfer requests linked to units. |
| 5 | **WhatsApp Integration (Free)** | Built-in wa.me share links across all modules — no paid API. Pre-filled message templates for visitor invite, complaint update, maintenance reminder, vendor recommendation, announcement, emergency contact, and invoice reminder. Copy message / Copy link / QR code options. |
| 6 | **Vendor Reviews & Vendor Booking** | Tenants can browse vendor marketplace, book services, and write star ratings with comments. Admin can set/update vendor ratings directly. Tenants can recommend new vendors. Booking confirmation and cancellation flow. |
| 7 | **Announcements, Community Events & Emergency Contacts** | Announcements — admin publishes, tenants read with unread badge and glowing auto-rotating banner on dashboard. Community Events — create events, tenant registration, upcoming/past views, WhatsApp share. Emergency Contacts — admin manages, viewable by tenants and security. |
| 8 | **Business Ads & Ad Billing** | Super Admin creates and manages business ads visible to all roles. Ad packages with duration and pricing. Click tracking, impressions, CTR analytics. Billing per ad, renewal reminders, revenue analytics dashboard. |
| 9 | **Daily Operations, Expense Tracking, Inventory & Assets** | Daily Operations — day-to-day task logging. Office Expenses — expense logging by category with summaries. Inventory — item tracking with movement logs. Asset Tracking — individual asset tags, staff checkout/checkin, condition tracking, periodic audit logs. |
| 10 | **Reports — GST, Audit Log & Suspend List** | Accounts & Compliance — GST tracking and compliance reports. Audit Log — full system activity log. Suspend List — report of all suspended/inactive users. CSV and PDF export for all report types. |
| 11 | **AMC & DG Maintenance** | Annual Maintenance Contract management — contract tracking and renewal. Diesel Generator maintenance logs — service records, fuel logs, fault entries. Managed from Super Admin portal. |
| 12 | **Home Automation Integration** | Home Assistant integration for smart device control. Hub management per building. Tenant "My Home" page for personal device control. Real-time device status dashboard. |
| 13 | **CCTV Integration per Block** | Camera device registration per block/zone (Entrance, Parking, Lobby, Corridor, etc.). Camera health monitoring (Online/Offline/Maintenance/Fault). Event logging (Motion, Intrusion, Manual). Snapshot records. Event timeline per camera. RTSP/IP architecture ready for future live stream integration. |
| 14 | **Lift Fault Alert & Email Alerts** | Lift fault events logged via CCTV/IoT event system. Admin notified on critical events. Email alert infrastructure via MailService — ready for Lift fault, DG fault, and other critical system alerts. |
