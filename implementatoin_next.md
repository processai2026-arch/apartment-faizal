# Finance & Expense Management
● Office Expense Module: Create a dedicated page to handle petty cash and cheque
payments.
● Print Support: The expense module must support generating and printing reports in A5
size.
● Document Attachment: Ensure the system allows for digital copies (e.g., a xerox of the
back of a cheque) to be attached and maintained within the expense records.

# Facility & Daily Operations
● Document Maintenance: Implement a specific category for "Office Documents" storage
and maintenance.
● Tenant Management: Add a "Name Transfer" feature within the association module to
handle incoming tenants.
● Daily Activity Reports: Create automated reporting or entry forms for the following daily
checks:
○ Staff Attendance and Maintenance Schedules.
○ CCTV Daily Status (e.g., tracking the status of all 20 cameras).
○ Water Lorry delivery logs.
○ Vendor Payment reports.
○ Electricity (EB) and Housekeeping logs.

# Inventory & Utility Tracking
● Utility ID System: Develop a tracking system to monitor utility usage and maintenance
gear (e.g., tower cleaning equipment).
● Asset Assignment: The system must track the inventory of safety gear (seat belts,
shoes, hand gloves) and log which specific employee ID has checked out which item from
the available sets.

# IoT & Hardware Automation -> lift fault and other ting looking
● Automated Fault Detection: The system must integrate with facility hardware (Lifts,
Electrical Boards) to monitor for voltage fluctuations or board issues.
● Hardware Specifications: The integration requires reading data from a hardware
communication module equipped with 16 to 26 I/O lines, connected via LAN or Wi-Fi.
● Automated Alerts: Upon detecting a hardware issue, the system must automatically
dispatch an alert email directly to the association secretary and the internal panel.
